## üöÄ Deployment

### GitHub Pages (Recommended for Play Store Link)
1. **Push to GitHub**: 
   ```bash
   git init
   git add .
   git commit -m "Launch Nur AI"
   git remote add origin https://github.com/YOUR_USERNAME/nur-ai.git
   git push -u origin main
   ```
2. **Enable Pages**: Go to Repository Settings > Pages and select the `main` branch.
3. **Privacy Policy for Play Store**: 
   Use this URL for your Store Listing: `https://YOUR_USERNAME.github.io/nur-ai/?view=privacy`

### Play Store (Android)
- This app is PWA-ready. You can use **Bubblewrap** or **TWA (Trusted Web Activity)** to package this for the Google Play Store.
- Use the `manifest.json` for app configuration.
